Quelques définitions

HTML
L'Hypertext Markup language, généralement abrégé HTML est le format de données conçu pour présenter les pages web
CSS
Les feuilles de style cascade, généralement appeléés CSS de l'anglais Cascading Style Sheet, forment un language informatique qui décrit la présentation des documents HTML
PHP
PHP: Hypertext Preprocessor, plus connu sous son sigle PHP ( acronyme récursif), est un language de programmation libre principalement utilisé pour produire des pages Web dynamiques via un serveur HTTP
